**Accrue.Billing.Metadata** (`accrue/lib/accrue/billing/metadata.ex`):
- Module with `validate_metadata/2` function accepting a changeset and field name
- Enforces Stripe-compatible constraints per D2-07: max 50 keys, key max 40 chars, value max 500 chars, no nested maps, values must be strings
- `""` or `nil` value means "delete this key" — documented in moduledoc
- Exports `shallow_merge/2` for the update path: merges new keys into existing, `""`/`nil` deletes
- No deep merge (D2-10 explicitly rejects it)

**Accrue.Billing.Customer** (`accrue/lib/accrue/billing/customer.ex`):
- Schema table: `"accrue_customers"`
- Fields:
  - `id` :binary_id, primary_key, autogenerate
  - `owner_type` :string, null: false (per D2-02: explicit string, NOT `__MODULE__` coercion)
  - `owner_id` :string, null: false (per D2-01: string column, lossless for UUID/bigint/ULID)
  - `processor` :string, null: false (e.g. "stripe", "fake")
  - `processor_id` :string (Stripe customer ID like "cus_xxx")
  - `name` :string
  - `email` :string
  - `metadata` :map, default: %{} (Stripe-compatible flat string/string)
  - `data` :map, default: %{} (processor-owned cache)
  - `lock_version` :integer, default: 1 (D2-09 optimistic_lock)
  - timestamps
- `changeset/2` function: casts all fields, validates required [:owner_type, :owner_id, :processor], calls `Accrue.Billing.Metadata.validate_metadata(changeset, :metadata)`, uses `Ecto.Changeset.optimistic_lock(:lock_version)`
- `unique_constraint(:owner_id, name: :accrue_customers_owner_type_owner_id_processor_index)` for the composite unique

```elixir
create table(:accrue_webhook_events, primary_key: false) do
  add :id, :binary_id, primary_key: true, default: fragment("gen_random_uuid()")
  add :processor, :string, null: false
  add :processor_event_id, :string, null: false

create unique_index(:accrue_webhook_events, [:processor, :processor_event_id],

create index(:accrue_webhook_events, [:type])
create index(:accrue_webhook_events, [:livemode])
create index(:accrue_webhook_events, [:status],

  name: :accrue_webhook_events_failed_dead_index)
```

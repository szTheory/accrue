    ```elixir
    defmodule Accrue.Storage do
      @moduledoc """
      Behaviour facade for pluggable PDF/asset storage (D6-04).
      v1.0 ships `Accrue.Storage.Null` only — all three callbacks are
      no-ops. `Accrue.Storage.Filesystem` lands in v1.1. Hosts needing
      S3 write a custom adapter.
      """

      @type key :: String.t()
      @type meta :: map()

      @callback put(key(), binary(), meta()) :: {:ok, key()} | {:error, term()}
      @callback get(key()) :: {:ok, binary()} | {:error, term()}
      @callback delete(key()) :: :ok | {:error, term()}

      @spec put(key(), binary(), meta()) :: {:ok, key()} | {:error, term()}
      def put(key, binary, meta \\ %{}) when is_binary(key) and is_binary(binary) and is_map(meta) do
        adapter = impl()
        metadata = %{adapter: adapter, key: key, bytes: byte_size(binary)}

        Accrue.Telemetry.span([:accrue, :storage, :put], metadata, fn ->
          adapter.put(key, binary, meta)
        end)
      end

      @spec get(key()) :: {:ok, binary()} | {:error, term()}
      def get(key) when is_binary(key) do
        adapter = impl()
        metadata = %{adapter: adapter, key: key}

        Accrue.Telemetry.span([:accrue, :storage, :get], metadata, fn ->
          adapter.get(key)
        end)
      end

      @spec delete(key()) :: :ok | {:error, term()}
      def delete(key) when is_binary(key) do
        adapter = impl()
        metadata = %{adapter: adapter, key: key}

        Accrue.Telemetry.span([:accrue, :storage, :delete], metadata, fn ->
          adapter.delete(key)
        end)
      end

      @doc false
      def impl, do: Application.get_env(:accrue, :storage_adapter, Accrue.Storage.Null)
    end
    ```

|-----------|----------|-----------|-------------|-----------------|
| T-06-02-01 | DoS | `Accrue.PDF.Null` crashing Oban workers via raise | mitigate | Adapter returns tagged error tuple; Logger.debug only. D6-06 mandates no raise inside adapter — test asserts `render/2` is side-effect-free aside from log |
| T-06-02-02 | Information Disclosure | Telemetry metadata leaking PII (HTML body, binary bytes) | mitigate | Storage span metadata is `%{adapter, key, bytes}` only — byte_size scalar, no binary payload. PDF span metadata stays `%{size, archival, adapter}` (Phase 1 precedent) |
| T-06-02-03 | Path Traversal (future) | Storage.Null `put/3` accepts `key` untrusted | accept | v1.0 Null echoes key only, never touches filesystem. Add path-normalization guard in Filesystem adapter (v1.1 scope) |
| T-06-02-04 | Repudiation | `{:error, :not_configured}` silently swallowed by callers | mitigate | Callers (Billing.fetch_invoice_pdf in Plan 06) MUST propagate the error; not swallowed. Audit via grep at plan 06 verification |

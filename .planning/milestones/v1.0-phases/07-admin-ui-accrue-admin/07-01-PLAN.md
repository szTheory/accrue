| T-07-01-02 | Tampering | `AccrueAdmin.Assets` | mitigate | Serve compile-time bundled files via hash-suffixed paths; tests assert path hash matches file contents |
| T-07-01-03 | Elevation of Privilege | mount macro auth boundary | mitigate | Wire `on_mount` inside the library `live_session` now so page plans cannot bypass mount-time auth checks later |

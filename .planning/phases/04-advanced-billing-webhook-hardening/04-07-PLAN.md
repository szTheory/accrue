## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Host success-URL controller → `Accrue.Checkout.reconcile/1` | Host passes user-visible `checkout_session_id` from URL |
| `Accrue.BillingPortal.Session.url` | Short-lived bearer credential — anyone holding URL impersonates customer |
| Stripe webhook `checkout.session.completed` → default_handler | Signed payload verified upstream |

| T-04-07-02 | Tampering | Customer replays `checkout_session_id` to reconcile arbitrary state | mitigate | `reconcile/1` refetches from Stripe (WH-10 lock — Stripe canonical); local state only mirrors what Stripe returns |

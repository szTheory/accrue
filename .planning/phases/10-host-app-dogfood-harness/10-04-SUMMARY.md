## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] Guarded the generated REVOKE migration for local host-app test bootstrap**
- **Found during:** Task 1 (Run the public installer and keep the generated host wiring intact)
- **Issue:** `mix test` in `examples/accrue_host` failed during `ecto.migrate` because the generated `99999999999999_revoke_accrue_events_writes.exs` migration assumed a Postgres role named `accrue_app` already existed.
- **Fix:** Wrapped the host migration's `REVOKE` / `GRANT` statements in `to_regrole(...)` checks so local setup succeeds when that optional application role is absent.
- **Files modified:** `examples/accrue_host/priv/repo/migrations/99999999999999_revoke_accrue_events_writes.exs`
- **Verification:** `cd examples/accrue_host && mix test test/install_boundary_test.exs`
- **Committed in:** `9d74253` (part of task commit)

**2. [Rule 1 - Bug] Fixed installer router patch idempotence for formatted macro calls**
- **Found during:** Task 2 (Make the host user billable and finish the generated billing-facade proof)
- **Issue:** Re-running `mix accrue.install` duplicated the webhook pipeline and `/billing` mount because `Accrue.Install.Patches` only detected `accrue_webhook "/stripe", :stripe` / `accrue_admin "/billing"` and missed Phoenix-formatted `accrue_webhook("/stripe", :stripe)` / `accrue_admin("/billing")`.
- **Fix:** Updated installer patch detection to recognize both macro-call styles, restored the host router to a single webhook/admin block, and added occurrence-count assertions in the install-boundary proof.
- **Files modified:** `accrue/lib/accrue/install/patches.ex`, `examples/accrue_host/lib/accrue_host_web/router.ex`, `examples/accrue_host/test/install_boundary_test.exs`
- **Verification:** `cd examples/accrue_host && mix accrue.install --yes --billable AccrueHost.Accounts.User --billing-context AccrueHost.Billing --admin-mount /billing --webhook-path /webhooks/stripe && mix test test/install_boundary_test.exs test/accrue_host/billing_facade_test.exs`
- **Committed in:** `57ba5be` (part of task commit)

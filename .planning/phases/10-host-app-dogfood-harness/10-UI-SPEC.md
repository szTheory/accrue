# Phase 10 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | Phoenix.Component primitives with the existing `accrue_admin` Tailwind preset and package CSS tokens |
| Icon library | none; prefer text labels and native disclosure affordances over icon-only controls |
| Font | `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif` |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Eyebrow spacing, tight inline gaps, status-dot offsets |
| sm | 8px | Chip gaps, compact controls, inline action spacing |
| md | 16px | Default field spacing, card internals, list rhythm |
| lg | 24px | Section padding, grouped action blocks, modal content spacing |
| xl | 32px | Page-section gaps, shell gutters on larger screens |
| 2xl | 48px | Major block separation, empty-state breathing room |
| 3xl | 64px | Page-level separation for major host-app sections only |

Exceptions: 44px minimum touch target for icon-only or disclosure controls; existing `ax-button` and `ax-icon-button` patterns already satisfy this with 2.75rem height.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FAFBFC` light / `#0F1318` dark | Page background, shell background, large canvas surfaces |
| Secondary (30%) | `#FFFFFF` light / `#171D24` dark | Cards, sidebar, topbar, drawers, elevated containers |
| Accent (10%) | `#5D79F6` | Primary CTA, active nav/tab state, links, focus ring, theme-active controls, brand chip, chart highlight |
| Destructive | `#111418` | Refund/cancel/confirm-danger actions only, always paired with explicit confirmation or step-up |

Accent reserved for: primary subscription action, plan-swap highlight, authenticated admin replay/requeue CTA, selected navigation/tab state, inline links, focus ring, and runtime host-brand accent surfaces. Do not use accent as a generic fill for all badges, cards, or status colors.

Additional semantic colors remain fixed to the existing admin palette: success `#5E9E84`, warning `#C8923B`, neutral text/nav `#24303B`, muted borders `#E9EEF2`.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Start subscription |
| Empty state heading | No billing activity yet |
| Empty state body | Billing records appear after a user starts a subscription or a webhook is processed. Start a subscription or review the webhook feed. |
| Error state | We couldn't complete that billing action. Check auth, migrations, webhook signing, or processor setup, then try again. |
| Destructive confirmation | Cancel subscription: Confirm cancellation before ending access. Replay webhook: Replay this webhook row. Refund charge: Confirm the amount and source event before continuing, then require step-up verification. |

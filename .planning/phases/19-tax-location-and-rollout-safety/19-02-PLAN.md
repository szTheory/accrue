## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| public host call -> `Accrue.Billing.update_customer_tax_location/2` | Untrusted customer location input crosses into Accrue's public API |
| processor result -> local `Customer` row | Provider response is reduced into the persisted customer projection |

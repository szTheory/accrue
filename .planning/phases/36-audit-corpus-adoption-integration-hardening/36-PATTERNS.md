# Phase 36 — Patterns

**Single contributor entrypoint** for doc pins; failures should name **REQ** or **script section** where possible.

**Three-source matrix:** REQUIREMENTS checkboxes + `*-VERIFICATION.md` + `requirements-completed` in `*-SUMMARY.md` stay aligned; prefer **links** from `.planning/` to canonical runnable docs over duplicating long command tables.

**CI stability:** job **ids** immutable; human-readable `name:` and prose evolve in docs together.

# Quick Task 260425-gr1: Drop deprecated flat-branding-keys infrastructure

## Why

Accrue is unpublished pre-1.0 and has zero real users. There is no migration
path to support, so the deprecation shim, boot-time warning, accompanying
docs, and tests are dead weight. The nested `:branding` keyword list is the
only supported shape going forward.
